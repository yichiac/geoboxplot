# Usage
## Installation 
```
$ git clone https://github.com/yichiac/geoboxplot
$ cd geoboxplot 
$ python setup.py install
```
## Data format
### climate csv file
First column: 經度 

Second column: 緯度

Third column: 基期資料

Forth column: 未來投影資料

Fifth column: 變化量資料

### geo csv file
區域內的點位資料:

First column: 經度 

Second column: 緯度
